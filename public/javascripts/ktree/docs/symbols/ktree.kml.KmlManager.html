findAndLoadKml(datasourceName, uri)
Loads KML for the specified data source. If the data source has been previously loaded, the KML will be retrieved from a cache. If the data source is new, the method will attempt to load it from a URI. Once loaded the KML data source is processed and displayed.
Parameters:
{string} datasourceName
A unique identifier for this KML data source
{string} uri
A URI from which the KML for the data source can be retrieved
loadKmlFromString(dataSourceName, kmlString)
Loads KML for the specified data source from a string. Once loaded the KML data source is processed and displayed.
Parameters:
{string} dataSourceName
A unique identifier for this KML data source
{string} kmlString
A string representation of the KML to load, expected to be wrapped in double quotes.
<private> loadKmlFromUri_(datasourceName, uri)
Loads KML for the specified data source by retrieving it from a remote KML (.xml) file. Once loaded the KML data source is processed and displayed.
Parameters:
{string} datasourceName
A unique identifier for this KML data source
{string} uri
The URI for the KML (.xml) file to load
<private> sendKmlToWorld_(kmlString, parentName)
Transmit KML to the World for display. Note that this method does NOT check to see whether the World object's API is actually ready to receive KML. For that functionality, ktree.kml.KmlManager.delayedSendKmlToWorld_() should be used.
Parameters:
{string} kmlString
The string of KML to transmit to the World
{string} parentName
Optional. An identifier for the KML node to which the argument KML data should be parented
